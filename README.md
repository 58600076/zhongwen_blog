2026年3月4日

作者白谣节点

我从14岁开始接触电脑,算是一个老网民了,我见证过,从最初得信息港,到QQ,百度搜索,到优酷网,淘宝的上线,B站.智能手机带来的移动端,微信,等各种互联网大战.这一段,可能是废话,但我想说的是这一个背景,答案在下面,因他们是关联的.所以废话了一段.


## 中文博客的评价:
恶劣的情况下个人独立博客很难生存.广告热情是玩不起来了. 我收集了很多rss,但发现,实质上更新大多停在了2015前后.

少数坚持更新的很少了.一年难有几篇产出. 这也不能怪谁,因个人精力有限,且谈钱的社会,谁还能分享好内容呢.即便是分享了,也拿不到一分钱.注意是一分钱都没有. 搜索引擎很快会抓去,变成它的围墙内容.倒是你再看自己写的帖子,还得付费.

## 门槛已经降到了零点,
各种自媒体平台,现成的.简单易用.且都Ai自己写了.


## 带宽的加速
正如乔布斯所言,带宽限制了互联网的发展,如今,各种直播,视频,制作录制门槛大幅降低.就连围墙一哥,都不得已向某抖妥协,开放了近乎17,8年的互不打扰的命根子.


## 预测

80后的孩子们正走向社会,他们玩的贼溜,6岁写代码玩魔兽,14岁的up主比比皆是.

现实存量的市场,会倒闭更多人进入互联网平台,有Ai的加持,垃圾信息,会持续增加, 打造自己小圈子是必走之路. 大厂玩围墙,网友可玩更小的孤岛.这在web上叫做二级入口.
